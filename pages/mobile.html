Shadow Engine - Lua Script API

Mobile Functions

Functions for mobile controls, touch input, vibration, and Android-specific features.

mobileControlsMode

mobileControlsMode:String

Returns the current mobile controls mode.

  • "left"
  • "right"
  • "custom"
  • "hitbox"
  • "none" / "unknown"

Extra Buttons

extraButtonPressed(button:String):Bool

extraButtonJustPressed(button:String):Bool

extraButtonJustReleased(button:String):Bool

extraButtonReleased(button:String):Bool

Checks the state of the mobile extra buttons.

  • "default" – First extra button
  • "second" – Second extra button

vibrate

vibrate(?duration:Int, ?period:Int):Void

Triggers device vibration.

  • duration – Vibration duration in milliseconds.
  • period (Optional) – Delay before vibration starts.

Touch Pad

addTouchPad(DPadMode:String, ActionMode:String, ?addToCustomSubstate:Bool = false, ?posAtCustomSubstate:Int = -1):Void

Adds a Lua-controlled touch pad.

removeTouchPad():Void

Removes the current Lua touch pad.

addTouchPadCamera(?defaultDrawTarget:Bool = false):Void

Assigns a camera to the Lua touch pad.
